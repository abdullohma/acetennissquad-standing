Rumus Perhitungan Poin Hybrid Margin System

Total Poin = Win % + Diff Point + Game Point
1. Diff Point (Poin Selisih Skor):
Merupakan gabungan dari selisih gol:
(Total Diff × 0.2) + (Avg Diff × 2.0)
2. Game Point (Poin Keaktifan):
Poin Dasar + (Margin Akar × Faktor Share):
(Games × 0.18) + [ (±√|Selisih|) × 1.5 × (1 + Share Ratio) ]
